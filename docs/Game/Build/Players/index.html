val neighbours : ?f:(t -> t) -> Player.t -> t -> Neighbours.t

neighbours x ys returns the neighbours of x in ys, i.e., those indexed either side of x. Note: requires that x be in ys. Note: if there is only one valid neighbour, then both Neighbours.t.left and Neighbours.t.right will refer to the same Player.t. Note: raises NoNeighbour in the event that x would be it's own neighbours.

val replace_player_role_kind : Roles.Role.Kind.t -> Roles.Role.Kind.t -> bool Roles.Map.t -> t -> t -> Player.t

replace_player_role_kind old new map players exclude replaces the role of one player in players that is not in exclude with kind old is replaced with a new role with kind new. We use map to ensure that we don't introduce duplicate roles into the game, and to that we can't re-add a role that has been removed.
